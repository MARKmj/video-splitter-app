# 项目结构

```
video-splitter-app/
├── public/
│   └── index.html              # HTML 模板
├── src/
│   ├── electron/
│   │   └── index.js            # Electron 主进程（后端逻辑）
│   ├── App.js                  # React 主组件
│   ├── App.css                 # 样式文件
│   └── index.js                # React 入口文件
├── package.json                # 项目配置和依赖
├── README.md                   # 项目说明
├── INSTALL.md                  # 安装和使用指南
└── .gitignore                  # Git 忽略文件
```

## 技术架构

### 前端（React）
- **App.js**: 主应用组件，包含所有 UI 逻辑
- **App.css**: 现代化渐变样式设计
- **index.html**: HTML 模板

### 后端（Electron Main Process）
- **electron/index.js**:
  - 窗口管理
  - 文件系统操作
  - FFmpeg 视频处理
  - IPC 通信处理

## 核心功能模块

### 1. 文件选择模块
- 输入文件夹选择
- 输出文件夹选择
- 视频文件扫描和过滤

### 2. 视频处理模块
- 基于 FFmpeg 的视频分割
- 支持多种视频格式
- 使用 `-c copy` 快速分割（无重编码）

### 3. 进度管理模块
- 实时进度显示
- 批量处理状态跟踪
- 错误处理和报告

### 4. UI 交互模块
- 拖动滑块设置时长
- 预设按钮快速选择
- 视频列表多选
- 实时反馈和动画

## 设计亮点

1. **渐变主题**: 紫色渐变背景，现代感十足
2. **卡片式布局**: 清晰的模块划分
3. **微交互动画**: 悬停效果、进度动画
4. **响应式设计**: 适配不同屏幕尺寸
5. **无重编码**: 使用 `-c copy` 快速分割视频

## 性能优化

1. **硬件加速**: FFmpeg 使用 GPU 加速
2. **批量处理**: 队列式处理多个视频
3. **实时反馈**: IPC 通信实时更新进度
4. **内存管理**: 流式处理大文件

## 扩展性

项目采用模块化设计，可以轻松扩展：
- 添加更多视频处理功能（压缩、转码等）
- 支持更多输出格式
- 添加视频预览功能
- 支持拖放文件操作
