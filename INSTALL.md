# 快速开始指南

## 1. 安装依赖

在终端中进入项目目录并安装依赖：

```bash
cd video-splitter-app
npm install
```

## 2. 运行应用

### 开发模式（推荐首次使用）
```bash
npm run dev
```

这将启动 React 开发服务器和 Electron 窗口。

### 生产模式
```bash
npm run build
npm start
```

## 3. 使用说明

1. **选择输入文件夹**：点击"选择文件夹"按钮，选择包含视频文件的文件夹
2. **选择输出文件夹**：选择分段后视频的保存位置
3. **设置分段时长**：
   - 拖动滑块调整时长（10-600秒）
   - 直接输入数字
   - 或点击预设按钮（30秒、60秒、120秒、180秒、300秒）
4. **选择视频**：在视频列表中勾选需要处理的视频
5. **开始处理**：点击"开始分段"按钮

## 功能特点

- ✨ 现代化 UI 设计，渐变紫色主题
- 🎬 支持多种视频格式（MP4, AVI, MOV, MKV, FLV, WMV, WEBM）
- ⚡ 快速处理，使用 FFmpeg 硬件加速
- 📊 实时进度显示
- 🎯 精确控制分段时长
- 💾 批量处理多个视频

## 打包成桌面应用

### macOS
```bash
npm run dist
```
打包后的 .dmg 文件将在 `dist` 目录中。

### Windows
```bash
npm run dist
```
打包后的 .exe 文件将在 `dist` 目录中。

## 注意事项

- 确保有足够的磁盘空间存储分段后的视频
- 处理大文件时可能需要较长时间
- 输出文件夹会自动创建分段文件（文件名格式：原文件名_01.扩展名）

## 常见问题

**Q: 为什么安装依赖很慢？**
A: 可以使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`

**Q: FFmpeg 相关错误？**
A: 重新安装依赖：`rm -rf node_modules && npm install`

**Q: 应用无法启动？**
A: 检查 Node.js 版本（需要 14+），更新到最新版本
